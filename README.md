# BYU-Pathway Worldwide Online

## WDD 330 - Web Frontend Development II

# ⛺ SleepOutside Starter Code

This repository contains the **SleepOutside** web application project for **WDD 330 – Web Frontend Development II**. The project is completed incrementally through team and individual assignments during the course.

Course reference:
https://byui-cse.github.io/wdd330-ww-course/week01/team.html

## Prerequisites

Before running this project, ensure you have the following installed:

* **Node.js** (LTS version recommended)
* **npm** (included with Node.js)

Setup guide:
https://byui-cse.github.io/wdd330-ww-course/intro/

## Installation

After cloning the repository, install the project dependencies:

```bash
npm install
```

> **Important:** The project will not run correctly until all dependencies have been installed.

## Running the Project

Start the local development server:

```bash
npm run start
```

The application will be available in your browser at the local URL displayed by Vite (typically `http://localhost:5173`).

## Common Workflow Commands

Run ESLint to check for code quality issues:

```bash
npm run lint
```

Automatically format the project using Prettier:

```bash
npm run format
```

Start the local development server:

```bash
npm run start
```

Build the production version of the project:

```bash
npm run build
```

## Troubleshooting

If you receive errors such as:

* `'vite' is not recognized`
* `'eslint' is not recognized`
* `Cannot find module 'prettier'`

run:

```bash
npm install
```

to install all required dependencies before running the application.
